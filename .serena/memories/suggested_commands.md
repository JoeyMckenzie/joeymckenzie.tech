Setup: `composer install && npm install && npm run build` (or `composer setup`). Dev loop: `composer dev` (runs artisan serve, queue listener, pail, npm dev). Frontend-only: `npm run dev` / `npm run build`. Tests: `composer test` (clears config + runs artisan test --parallel), `php artisan test --filter=...` for targeted cases. Static analysis/refactor: `composer lint` (Larastan), `composer refactor` (Rector), `composer typos` (Peck). Formatting: `vendor/bin/pint --dirty` (or `composer fmt`). IDE helpers: `composer generate:ide-helpers`. Resync DB/schema: `composer reschema`.
