# joeymckenzie.tech

The source for my personal blog at [joeymckenzie.tech](https://joeymckenzie.tech), where I write about Laravel, PHP, nix, neovim, and whatever else I'm tinkering with.

Built with Laravel, Inertia, and React. Posts live as markdown files in [`content/posts/`](content/posts) and are rendered through [Phiki](https://github.com/ryangjchandler/phiki) for syntax highlighting. Local development runs on nix + devenv (see [this post](https://joeymckenzie.tech/blog/local-laravel-with-nix) for the gory details).
