# joeymckenzie.tech

[![Laravel Forge Site Deployment Status](https://img.shields.io/endpoint?url=https%3A%2F%2Fforge.laravel.com%2Fsite-badges%2Fd7dd883c-231d-4186-b6da-e872797b053f&style=plastic)](https://forge.laravel.com/joey-mckenzie/polished-wildflower/3112152)

The source for my personal blog at [joeymckenzie.tech](https://joeymckenzie.tech), where I write about Laravel, PHP, random stuff, and whatever else I'm tinkering with.

Built with Laravel, Inertia, and React. Posts live as markdown files in [`content/posts/`](content/posts) and are rendered through [Phiki](https://github.com/ryangjchandler/phiki) for syntax highlighting. Local development runs on nix + devenv (see [this post](https://joeymckenzie.tech/blog/local-laravel-with-nix) for the gory details).
